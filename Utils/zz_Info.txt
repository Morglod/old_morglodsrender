Код из директории Utils содержит методы и интерфейсы общего назначения, имеет минимальные
зависимости и может с минимальными изменениями использоваться в других проектах.

#Containers - содержит набор различных контейнеров.
	StaticArray<T> - массив элементов типа T, не меняющий свой размер.

#Debug - содержит методы для упрощения отладки.
	Assert, AssertAndExec, AssertExec.

#Events - содержит набор различных классов, реализующих работу с событиями.
	Event<args> - класс, в качестве callback'ов вызывает статичные функции.
	TimerEvent<args> - Event<args> в связке с таймером. Вызывается по достижении опред времени. Должен обновляться методом Update(deltaTime).
	EventListener<args> - класс, в качестве callback'ов держит интерфейс класса, содержащий реализацию вызываемого метода.

#Exceptions - исключения для MorglodsRender

#Log - содержит класс лога, упрощающий обработку сообщений для отладки.
	Log, Log::Add(callback), Log::LogString.

#Time - надстройка над стандартной библиотекой времени. Реализация таймера.
	Timer<,>.
